### Ubiquitous Language

#### Nouns And Verb Collection

* Parse
* Command line arguments or argument list
* Utility
* Array of string to main() function
* Program
* Flags
  * Flags should be one character, preceded by a minus sign
* Values
  * Each flag should have zero, or one value associated with it
* Parser
* The schema
  * The schema specifies the number and types of flags and values the program expects
* Verify
* Match the schema
* Default value on missing flag value
  * “False” for a boolean
  * 0 for a number
  * "" for a string
* Error message
